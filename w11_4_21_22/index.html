What is it?

Markdown is a lightweight markup language. It allows you to add formatting and more complex elements to a plain text document.

Where you've probably seen it

You've probably seen markdown and didn't even realize it. The following places use markdown in some way:

  • Github
  • NPM
  • Stack Overflow
  • Wordpress
  • Other CMS platforms

Why to use it

We use markdown because it allows us to do complex things in a universal way. A markdown file can be used across various platforms, applications, and clients. So it makes your content portable and also re-usable. If you change platforms or update your frontend to use a new technology the markdown output will be the same regardless.
